# Governance Evaluator Bootstrap

This repository scaffold is for building the Phase 1 governance benchmark described in `TASK.md`.

## Start

1. Create a new Git repository from these files.
2. Open the repository root in Codex.
3. Send exactly:

   `Read AGENTS.md and execute TASK.md completely. Do not stop after planning.`

The first run must build and prove the benchmark only. It must not enable auto-merge, change branch protection, modify Spaghetti, or build the full autonomous orchestration system.

## Why the source documents are not enough by themselves

The documents under `docs/` describe intent and prior analysis. `AGENTS.md` establishes durable repository rules, while `TASK.md` gives Codex a bounded implementation mission and objective completion criteria.
